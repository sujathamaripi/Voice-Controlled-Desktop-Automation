🎙️ Voice-Controlled Desktop Automation

A Python-based Voice-Controlled Desktop Automation system that allows users to control their computer using voice commands. The assistant can open applications, search the web, send WhatsApp messages, control system settings, perform calculations, take screenshots, capture photos, and much more through simple voice interactions.

 📌 Features

- 🎤 Voice Recognition using SpeechRecognition
- 🔊 Text-to-Speech responses using pyttsx3
- 🌐 Open websites (Google, YouTube, Gmail, ChatGPT)
- 🔍 Search anything on Google
- ▶️ Play YouTube videos
- 💬 Send WhatsApp messages to saved contacts
- 📷 Open camera and capture photos
- 📸 Take screenshots
- 📝 Typing mode using voice
- 💻 Open desktop applications
  - Chrome
  - Notepad
  - Calculator
  - Command Prompt
  - VS Code
  - Microsoft Word
  - Microsoft Excel
  - Microsoft PowerPoint
  - File Explorer
  - WordPad
- 🔒 Lock computer
- 🌙 Sleep system
- 🔄 Restart system
- ⚡ Shutdown system
- 🔊 Volume control
- ☀️ Screen brightness control
- ⏰ Current time announcement
- 🌦️ Live weather updates
- 🧮 Voice calculator
- 😂 Tell jokes
- 💪 Motivation quotes
- ❌ Close running applications
- 🖥️ Simple Tkinter GUI

 📦 Required Libraries

Install all required packages using:

```bash
pip install SpeechRecognition
pip install pyttsx3
pip install pywhatkit
pip install pyautogui
pip install requests
pip install screen-brightness-control
pip install opencv-python
pip install pyaudio
```

 ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Voice-Controlled-Desktop-Automation.git
```

2. Navigate to the project folder

```bash
cd Voice-Controlled-Desktop-Automation
```

3. Run the application

```bash
python assistant.py
```

4. Click **Start Listening** and speak your commands.
